# Photo Curator User Prompt — 第一轮：选片 + 排版

> PhotoArtist 在第一轮 sessions_spawn 时，将此 Prompt 与缩略图一起发送给 PhotoCurator。配合 `photo-curator-system-prompt.md` 使用。

---

````

我拍了一组照片（见附件中的缩略图），请你帮我完成以下两件事：

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 一、选片

从这批照片中挑选出最适合发布的 **9 张**（如果总数不足 9 张则全选），并为每张入选照片说明理由：
- 构图亮点
- 光影 / 色彩表现
- 情绪 / 故事性
- 淘汰照片的简要原因（可合并说明）

请用表格输出：

| 序号 | 文件名 | 入选理由 |
|------|--------|----------|

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 二、排版方案 + 风格定义

请分别针对以下两个平台给出排版建议：

### 📱 微信朋友圈
- 推荐发几张（1/2/3/4/6/9 宫格）
- 图片排列顺序（第一张最重要，会作为封面）
- 配文（直接给出可用的成品文案）

### 📕 小红书
- 推荐发几张（封面 + 内页）
- 封面图选择 & 封面文字标题
- 内容排列逻辑（叙事顺序 / 对比 / 氛围递进等）
- 标题 + 正文 + 标签（直接给出可发布的成品）

### 🎨 风格意图

请确定这组照片的整体视觉方向，供后续调色使用：
- **风格方向**：如"暖春清新""电影胶片""莫兰迪低饱和"等
- **氛围关键词**：如"轻快、温暖、有生活气息"
- **一致性模式**：`unified`（统一风格）/ `contrast`（对比编排）/ `gradient`（渐变过渡）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

## 输出要求

1. 使用中文，Markdown 格式
2. 顺序：选片总览 → 排版方案 → 风格意图
3. 配文和标题是**成品**，不是"方向建议"
4. **⚠️ 必须在最后输出 `layout_config` JSON**

### ⚠️ layout_config JSON（必须输出）

```json
{
  "style_intent": {
    "direction": "风格方向名称",
    "mood": "氛围关键词",
    "consistency": "unified",
    "notes": "特殊说明"
  },
  "wechat": {
    "count": 9,
    "order": ["DSC_0001.NEF", "..."],
    "caption": "朋友圈配文成品"
  },
  "xiaohongshu": {
    "count": 9,
    "cover": "DSC_0001.NEF",
    "order": ["DSC_0001.NEF", "..."],
    "title": "小红书标题",
    "body": "小红书正文",
    "tags": ["#标签1", "#标签2"]
  }
}
```

- `order` 中的文件名必须与选片表格中的文件名一致
- `style_intent` 将传递给第二轮调色任务
- 这个 JSON 将被保存为 `layout_config.json`

````
